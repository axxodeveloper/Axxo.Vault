## Overview

- **Axxo Vault** is a modern Android application designed for fast, secure, and seamless file sharing across local networks.

- Built entirely with **Kotlin** and **Jetpack Compose**, Axxo Vault enables direct peer-to-peer transfers and browser-based file access without relying on cloud storage, third-party servers, or internet connectivity.

- Whether devices are connected through a Wi-Fi router or a mobile hotspot, files can be shared instantly with minimal setup.

---

## Features

### Lightning-Fast File Transfers

- Send and receive files directly between devices connected to the same local network.

### Built-In Web Sharing Portal

- Access shared files from any browser using a simple local URL.

### No Internet Required

- Works completely offline using local network communication.

### Privacy First

- Files remain within your network and are never uploaded to external servers.

### Modern Android Experience

- Crafted using Jetpack Compose and Material Design 3.

### Edge-to-Edge UI

- Immersive layouts designed for modern Android devices.

### Responsive & Lightweight

- Optimized for smooth performance and efficient resource usage.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AxxoVault.git
```

Navigate to the project:

```bash
cd AxxoVault
```

Build the application:

```bash
./gradlew assembleDebug
```

Run on an Android emulator or physical device.
